Ejercicio 8.12

  • Suponga que i = 1, j = 2, k = 3 y m = 2. ¿Qué es lo que imprime cada una de las siguientes instrucciones? ¿Son necesarios los paréntesis en cada caso?

    • Ejercicio 12.

    1. document.writeln( i == 1 );

      2. Imprime: True


    2. document.writeln( j == 3 );

      3. Imprime: False


    3. document.writeln( i >= 1 && j < 4 );

      4. Imprime: True


    4. document.writeln( m <= 99 && k < m );

      5. Imprime: True


    5. document.writeln( j >= i || k == m );

      6. Imprime: True


    6. document.writeln( k + m < j || 3 – j >= k );

      7. Imprime: False


    7. document.writeln( !( k > m ) );

      8. Imprime: True
