Ejercicio de Autoevaluación 7.1

  • Complete los siguientes enunciados.

    1. Todas las secuencias de comandos pueden escribirse en términos de tres tipos de estructuras de control: secuencia, seleccion y repeticion

    2. La instrucción de selección doble if-else se utiliza para ejecutar una acción cuando una condición es verdadera, y otra accion cuando esa condición es falsa.

    3. Al proceso de repetir un conjunto de instrucciones un número específico de veces se le llama repetición controlada por contador (o definida)

    4. Cuando no se sabe de antemano cuántas veces se repetirá un conjunto de instrucciones, se puede usar un valor centinela ( o un de señal, de bandera o de prueba) para terminar la repetición.
