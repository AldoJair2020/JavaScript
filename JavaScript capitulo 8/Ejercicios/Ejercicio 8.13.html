  • Dada la siguiente instruccion switch:

    • switch (k)
    {
    case 1:
    break;
    case 2:
    case 3:
    ++k;
    break;
    case 4:
    --k;
    break;
    default:
    k *= 3;
    }
    x = k;

    ¿Qué valores se asignan a x cuando k tiene vaores de 1, 2, 3. 4 y 10?
