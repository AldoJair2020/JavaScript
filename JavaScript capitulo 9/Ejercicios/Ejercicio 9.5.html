Ejercicio de Autoevaluación 9.5

  • Localice el error en cada uno de los siguientes segmentos de programa y explique cómo corregirlo:

    1. method g()
      {
      document.writeln( "Dentro del método g")
      }

      Error: Method no es la palabra que se utiliza para comenzar la definición de una función.
      Corrección: Cambiar method por function.

    2. //Esta función debe devolver la suma de sus argumentos
      function suma(x,y)
      {
      var resultado;
      resultado = x + y;
      }

      Error: La función no devuelve algún resultado.
      Corrección: Agregar return resultado; al final.

    3. function f ( a )
      {
      document.writeln( a ); }

      Error: El punto y coma después del paréntesis derecho.
      Corrección: Eliminar el punto y coma que está después del paréntesis derecho.
