Ejercicio de Autoevaluación 8.3

  • Encuentre el error en cada uno de los siguientes segmentos de código, y explique cómo corregirlo:

    1. x = 1;
      while (x <= 10);
      ++x;
      }

      Error: El punto y coma después del encabezado while provoca un ciclo infinito y falta una llave izquierda.
      Correcion: Reemplazar el punto y coma por una llave izquierda o eliminar el punto y coma junto con la llave derecha.

    2. switch ( n )
      {
      case 1:
      document.writeln( "El número es 1" );
      case 2:
      document.writeln( "El número es 2" );
      break;
      default:
      document.writeln( "El número no es 1 ni 2" );
      break;
      }

      Error: Falta una instrucción break en el primer case.
      Correcion: Agregar break al final del case 1.

    3. n = 1;
      while (n < 10)
      document.writeln( n++);

      Error: Se está usando un operador relacional inadecuado después del while.
      Corrección: Usar <= en vez de <.
