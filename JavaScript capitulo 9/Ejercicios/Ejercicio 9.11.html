Ejercicio 9.11

  • Escriba instrucciones que asignen enteros aleatorios a la variable n en los siguientes rangos

    1. 1 ≤ n ≤ 2

      2. Respuesta: var NumR=Math.floor(Math.random()*2);


    2. 1 ≤ n ≤ 100

      3. Respuesta: var NumR=Math.floor(Math.random()*101);


    3. 0 ≤ n ≤ 9

      4. Respuesta: var NumR=Math.floor(Math.random()*10);


    4. 1000 ≤ n ≤ 1112

      5. Respuesta: var NumR=Math.floor(Math.random()*(1112-1000)+1000);


    5. –1 ≤ n ≤ 1

      6. Respuesta: var NumR=Math.floor(Math.random()*(1-(-1))-1);


    6. –3 ≤ n ≤ 11

      7. Respuesta: var NumR=Math.floor(Math.random()*(11-(-3))-3);
