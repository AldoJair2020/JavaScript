Ejercicio 8.4

  • Encuentre el error en cada uno de los siguientes segmentos de código. [Nota: puede haber más de un error].

    1. For ( x = 100, x >= 1, ++x )
      document.writeln( x );

      2. Error: no se inicializó con "{" despues del "()" y For es con minuscula "for".

    2. El siguiente código debe imprimirse sin importar si el valor entero es par o impar:
      switch ( valor % 2 )
      {
      case 0:
      document.writeln( "Entero par" );
      case 1:
      document.writeln( "Entero impar" );
      }

      3. Error: el codigo para imprimir algo no es el optimo y a los case les falta el "break" y el opcional "default".


    3. El siguiente código debe imprimir los enteros impares del 19 al 1:
      for ( x = 19; x >= 1; x += 2 )
      document.writeln( x );

      4. Error: la variable x va en incremento, deberia ir en decremento. y está imprimiendo todos los numeros sin excepcion, cuando deberian ser solo los impares.


    4. El siguiente código debe imprimir los enteros pares del 2 al 100:
      contador = 2;
      do {
      document.writeln( contador );
      contador += 2;
      } } While ( contador < 100 );

      5. Error: el contador no llega a la meta (100) deberia ser while(contador<=100);
