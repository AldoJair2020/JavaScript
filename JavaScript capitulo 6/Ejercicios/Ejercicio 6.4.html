Ejercicio de Autoevaluación 6.4

  • Indentifique y corrija los errores en cada una de las siguientes instrucciones

    1. if (c < 7);
      window.alert("c es menor que 7");

        2. Error: No debe de haber punto y coma despues de condificional if

    2. if (c => 7)
      window.alert("c es mayor que 7");

        3. Error: El operador relacional => es incorrecto, por lo que se debe cambiar por >=
